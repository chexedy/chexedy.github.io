Fishing Frenzy

This is a simple 2-player Fishing Game I made for a club at my university. The original game is between 2-4 players, each on controller. The goal is to capture as many fish as you can before the clock runs out. I customized this version to be only two players, with keyboard controls.

Credit to my two partners for the game, one did the pixel art and the other did the main menu/results screen music.




Controls (Fullscreen Recommended)

Player 1 uses WASD and E

Player 2 uses the Arrow Keys and P



Please let me know on GitHub if you encounter any bugs
